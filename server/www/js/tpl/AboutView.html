Version 0.9.3 - ChangeLog

  • Medcom Onyx: improved support.
  • Fried Circuits OLED backpack: two-click firmware download/upgrade.

Version 0.9.2 - ChangeLog

  • New IoT service: Pinocc.io.
  • New Instrument: simple serial logger.

Version 0.9.0 - Changelog

  • First release version
  • Support for multiple open devices in Server mode.
  • Server mode uses PouchDB as storage engine.

Version 0.8.1 - Changelog

  • Firmware uploader support for some instruments.

Version 0.8.0 - Changelog

  • Chrome application now uses indexeddb for large capacity storage
  • Log management is cleaner - avoid deleting log used for recording, update start/end date properly
